Why Refity

A lightweight, secure way to run your own Docker registry. Use Hetzner Storage Box, your VPS, or any SFTP—you stay in control.

SFTP backend

Blobs and manifests live on your SFTP server. No proprietary storage—works with any provider.

Docker compatible

Standard docker push and docker pull. Works with existing tools and CI.

Web UI

Browse groups, list tags, copy pull commands. Responsive and simple.

JWT auth

Token-based auth. Optional Hetzner usage dashboard; works without it.

Quick start

Clone, set your SFTP credentials in .env, and run with Docker Compose.

1

Deploy Refity and point it at your SFTP server.

2

Create groups in the web UI (e.g. myteam).

3

Push with docker push your-registry/myteam/app:latest.

4

Pull from anywhere. List tags and copy commands in the UI.
